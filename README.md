# PT&L Equipment Services — website

Static site. To publish with **GitHub Pages**:

1. Push this repo to GitHub.
2. Repo **Settings → Pages**.
3. Source: **Deploy from a branch** → Branch **main** → Folder **/docs** → Save.
4. Your site goes live at `https://<user>.github.io/<repo>/`.

`index.html` is the homepage. All pages, scripts (`support.js`, `image-slot.js`), `assets/` and `uploads/` are included. An internet connection is required (React and Google Fonts load from a CDN).
