# PT&L Equipment Services — website

Static site, ready for **GitHub Pages**.

## Publish

1. Push this repo to GitHub.
2. Repo **Settings → Pages**.
3. Source: **Deploy from a branch** → Branch **main** → Folder **/docs** → Save.
4. Your site goes live at `https://<user>.github.io/<repo>/`.

## Pages

`index.html` (home), `about.html`, `services.html`, `areas.html`,
`melbourne-west.html`, `hallam.html`, `blog.html`, `post.html`,
`contact.html`, `book.html`.

`support.js`, `image-slot.js` and the `uploads/` images are all included.
An internet connection is required (React and Google Fonts load from a CDN).
